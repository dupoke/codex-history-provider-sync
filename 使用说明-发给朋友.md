# Codex History Provider Sync 使用说明

这是一个 Windows 小工具，用来修复 Codex Desktop 本地历史还在、但侧边栏看不到旧对话的情况。

## 怎么用

1. 解压整个压缩包。
2. 先暂停 Codex Desktop 里正在运行的任务；完全退出后再同步更稳。
3. 双击 `Start-Codex-History-Provider-Sync.cmd`。如果想用旧入口，也可以双击 `Start-Codex-History-Sync.cmd` 或 `双击启动.cmd`。
4. 打开后先看状态，再点“同步历史归属”。
5. 同步完成后重新打开 Codex Desktop。

## 需要什么环境

- Windows
- Python 3.10 或更高版本
- 本机已经安装并使用过 Codex Desktop

如果双击后提示找不到 Python，请安装 Python 3：

https://www.python.org/downloads/windows/

## 安全说明

- 同步前会自动备份本机 Codex 数据库、会话索引和会话元数据。
- 备份默认保存在 `%USERPROFILE%\.codex\history_sync_backups`。
- 如果结果不对，可以在工具里选择备份恢复。

## 不适合的场景

- 云端账号之间同步聊天记录
- 本地历史文件已经被删除
- 不同电脑之间迁移历史记录
